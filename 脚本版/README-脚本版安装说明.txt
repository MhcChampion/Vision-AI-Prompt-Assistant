Vision AI 提示词助手 - 脚本版说明

这个版本不会影响你原来的上架版文件。
我已经把脚本、安装器和原扩展文件都放进这个“脚本版”目录里了。

给用户怎么用：
1. 先把整个“脚本版”文件夹发给用户。
2. 用户解压后，双击“ 一键安装.bat ”。
3. 安装器会自动完成下面这些事：
   - 自动查找 Google Chrome
   - 自动把扩展复制到本机目录
   - 自动创建桌面快捷方式
   - 自动启动带扩展的 Chrome

安装后怎么打开：
- 用户以后只需要双击桌面的 “Vision AI 提示词助手” 快捷方式。
- 这个快捷方式会打开一个专门给你这个扩展准备的 Chrome 配置，不需要手动去浏览器里开启开发者模式。

更新脚本版的方法：
- 你改完脚本版里的文件后，让用户重新运行一次 “一键安装.bat” 即可覆盖更新。

卸载方法：
- 双击“卸载扩展.bat”。

注意：
- 这个方案适合在 Chrome 商店审核还没通过时给用户临时使用。
- 它不会改动你当前这个原项目目录。
- 如果用户电脑里没有安装 Google Chrome，安装会提示失败。
