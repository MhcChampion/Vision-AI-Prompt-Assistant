# Vision-AI-Prompt-Assistant

一个面向视觉设计逆向分析的 Chrome 扩展。

它会在网页图片上提供悬浮分析入口，也支持右键菜单触发；分析结果会同时输出中文和英文版本，并且可以在弹窗内直接复制、编辑和重做。

## 功能特性

- 悬浮按钮分析：鼠标移到图片上即可快速触发分析
- 右键菜单分析：支持对图片使用“视觉设计逆向分析”
- 双语输出：自动返回中文与英文两版结果
- 结果可编辑：分析完成后可在弹窗内直接修改文本
- 一键重做：基于同一张图重新生成另一版更细致的分析
- 多接口支持：支持 Gemini 官方、Gemini 兼容接口、火山方舟
- 快捷键触发：可在设置页自定义快捷键
- 日志查看：设置页内可查看最近成功日志和错误日志
- 额度查询：弹窗内支持第三方 Gemini 兼容代理接口额度查询

## 安装方式

### 本地加载

1. 下载或克隆本项目到本地
2. 打开 Chrome，进入 `chrome://extensions/`
3. 打开右上角“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择项目根目录

## 使用方法

1. 安装扩展后，点击扩展图标进入弹窗
2. 打开“设置”页填写接口配置
3. 在任意网页将鼠标移动到图片上
4. 点击“图像逆向分析”，或使用右键菜单触发
5. 等待返回结果后，可切换中英文、复制内容或重做一版

## 接口说明

### 1. Gemini 官方

- 在设置页填写 Gemini API Key
- 默认调用 `gemini-1.5-pro-latest`

### 2. Gemini 兼容接口

- 填写兼容接口地址和 API Key
- 可手动输入模型名
- 也可直接从接口拉取模型列表
- 弹窗中的额度查询仅对这一类代理接口生效

### 3. 火山方舟

支持两种模式：

- 按 Token 计费
- Coding Plan 会员

其中按 Token 计费模式使用 `Endpoint ID`，不是基础模型名。


## 注意事项

- 接口密钥与个性化配置保存在浏览器的 `chrome.storage.local` 中
- 第三方兼容接口的可用能力与额度查询结果取决于对应服务商实现
- 建议在正式使用前先完成设置页配置，并进行一次实际图片测试
